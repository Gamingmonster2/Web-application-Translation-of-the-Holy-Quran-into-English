# 📖 The Holy Quran: Interactive English Translation Web App

![Status: Live](https://img.shields.io)
![Technology: HTML/CSS/JS](https://img.shields.io)
![License: MIT](https://img.shields.io)

تطبيق ويب تفاعلي لعرض معاني القرآن الكريم باللغة الإنجليزية. تم تطوير هذا المشروع عبر تحويل صفحات المصحف من نسخة PDF عالية الجودة إلى صور محسنة، ودمجها في واجهة مستخدم سلسة وتفاعلية تتيح القراءة والتصفح بسهولة عبر المتصفح.

---

## 🌟 مميزات التطبيق (Key Features)

- **تجربة قراءة رقمية**: تصفح القرآن وكأنك تقرأ من المصحف الورقي بفضل تقنية الصور عالية الدقة.
- **ترجمة مدمجة**: النصوص الإنجليزية مرافقة للآيات لضمان الفهم الصحيح للمعاني.
- **تصميم متجاوب (Responsive)**: يعمل التطبيق بكفاءة على كافة الشاشات (هواتف، أجهزة لوحية، حواسيب).
- **سرعة التحميل**: تم تحسين الصور لضمان تحميل سريع دون استهلاك كبير للبيانات.
- **واجهة بسيطة**: تصميم عصري يركز على المحتوى لتوفير تجربة قراءة مريحة للعين.

## 🛠️ البناء التقني (Built With)

- **Frontend**: HTML5, CSS3 (Flexbox/Grid).
- **Interactivity**: JavaScript لتقليب الصفحات والتنقل.
- **Assets**: صور عالية الجودة محولة من النسخة الأصلية للـ PDF.
